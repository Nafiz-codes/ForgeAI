# ForgeAI Dataset Profile Specification

Version: 1.0

---

## Purpose

The Dataset Profile is a structured summary of a dataset generated by Python before any AI reasoning takes place.

Instead of sending raw CSV data to the language model, ForgeAI extracts statistical and structural information and sends only the profile. This reduces token usage, improves reasoning quality, and makes the system deterministic.

The AI never directly reads the dataset.

Workflow:

CSV
→ Dataset Profiler (Python)
→ Dataset Profile (JSON)
→ AI Reasoning
→ Preprocessing Plan

---

# Dataset-Level Information

These describe the dataset as a whole.

| Field | Description |
|--------|-------------|
| Dataset Name | Name of uploaded dataset |
| Rows | Total number of rows |
| Columns | Total number of columns |
| Duplicate Rows | Number of duplicated rows |
| Missing Cells | Total missing values |
| Memory Usage | Dataset size in MB |
| Possible ML Task | Classification / Regression / Unknown |
| Possible Target Column | AI's inferred target variable |

Example

```json
{
  "dataset_name": "Titanic",
  "rows": 891,
  "columns": 12,
  "duplicates": 3,
  "missing_cells": 177,
  "memory_usage": "0.42 MB",
  "possible_task": "classification",
  "possible_target": "Survived"
}
```

---

# Column-Level Information

Every column will be summarized independently.

## Common Fields

| Field | Description |
|--------|-------------|
| Name | Column name |
| Data Type | Numeric / Categorical / Boolean / Date |
| Missing Percentage | Missing value percentage |
| Unique Values | Number of unique values |
| Possible Identifier | True if values appear unique |

---

## Numeric Columns

Additional statistics:

- Mean
- Median
- Standard Deviation
- Minimum
- Maximum
- Skewness
- Outlier Count

Example

```json
{
  "name": "Age",
  "dtype": "numeric",
  "missing_percent": 19.8,
  "unique_values": 88,
  "mean": 29.7,
  "median": 28,
  "std": 14.5,
  "min": 0.42,
  "max": 80,
  "skewness": 0.78,
  "outliers": 12
}
```

---

## Categorical Columns

Additional statistics:

- Cardinality
- Most Frequent Value
- Frequency Distribution

Example

```json
{
  "name": "Embarked",
  "dtype": "categorical",
  "missing_percent": 0.2,
  "unique_values": 3,
  "cardinality": "Low",
  "most_common": "S"
}
```

# Design Philosophy

ForgeAI profiles datasets before AI reasoning.

Python extracts facts.

The AI interprets those facts.

The AI never directly manipulates the dataset.

Python performs all preprocessing after the AI has produced an approved preprocessing plan.
